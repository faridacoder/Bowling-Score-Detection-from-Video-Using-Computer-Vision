# Bowling Score Detection from Video Using Computer Vision

A computer vision system for detecting, tracking, and scoring bowling pins from recorded videos using OpenCV and deep learning.

## Features
- Bowling pin detection using YOLO
- Object tracking
- Automatic score calculation
- Video processing pipeline
- Deep learning based detection

## Technologies
- Python
- OpenCV
- YOLOv8
- Roboflow
- Ultralytics
